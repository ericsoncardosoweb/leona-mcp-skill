## O que é

Uma skill para o seu assistente de IA (Cursor, Claude e outros).  
Com ela + o **[Leona Flow](https://leonaflow.com)**, você pede em português e a automação WhatsApp é criada na **sua** conta.

Serve para qualquer negócio: loja, delivery, curso, consultoria, clínica, ofertas e mais.

---

## Instalar (só isso)

1. Tenha conta no **[Leona Flow](https://leonaflow.com)**  
2. Com o **MCP Leona** já conectado no seu editor  
3. Cole no chat do Agent:

```text
Instale a skill leona-flow a partir deste repositório:
https://github.com/ericsoncardosoweb/leona-mcp-skill
```

Pronto. O próprio editor instala no lugar certo.

### Usar

```text
/leona-flow
```

Ou: *“Siga a skill leona-flow e monte [seu objetivo].”*

Exemplos:

- *“Boas-vindas com menu e transferência para humano”*  
- *“Checkout com PIX e confirmação de pagamento”*  
- *“Follow-up se a pessoa não responder”*  

Depois abra o Leona, atualize o canvas e teste.
